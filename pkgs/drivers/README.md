# Proprietary Driver Packages for CIX CD8180/CD8160 (Sky1) SoC

## Available Drivers

### GPU Driver (`cix-gpu-umd`)
Mali-G610 MP4 GPU userspace drivers

### NPU Driver (`cix-npu-umd`)
Neural Processing Unit drivers (28.8 TOPS)

### ISP Driver (`cix-isp-umd`)
Image Signal Processor userspace drivers

### VPU Firmware (`cix-vpu-firmware`)
Video Processing Unit firmware

### System Firmware (`cix-firmware`)
Firmware for GPU, NPU, ISP, VPU, WiFi, Bluetooth

## Source

All drivers are packaged from [orangepi-xunlong/component_cix-current](https://github.com/orangepi-xunlong/component_cix-current).

## Usage

Drivers are automatically installed when using the Orange Pi 6 Plus board module.
